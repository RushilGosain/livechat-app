# LiveChat — Real-time Chat App

Built with **Next.js 14**, **Convex**, **Clerk**, **ShadcnUI**, and **Tailwind CSS**.

---

## Features

- ✅ Auth (Clerk — email + social login)
- ✅ User list & real-time search
- ✅ 1-on-1 direct messages
- ✅ Group chats
- ✅ Real-time messages (Convex subscriptions)
- ✅ Smart timestamps (Today / Feb 15 / Feb 15, 2023)
- ✅ Online/offline status
- ✅ Typing indicators
- ✅ Unread message counts
- ✅ Auto-scroll + "New messages" button
- ✅ Soft-delete messages ("This message was deleted")
- ✅ Emoji reactions (👍 ❤️ 😂 😮 😢)
- ✅ Skeleton loaders
- ✅ Empty states
- ✅ Responsive layout (mobile + desktop)
- ✅ Dark / Light mode

## Setup

### 1. Install dependencies

```bash
npm install convex @clerk/nextjs next-themes date-fns clsx tailwind-merge lucide-react
npm install tailwindcss@^3.4.0 autoprefixer postcss tailwindcss-animate
```

### 2. Install ShadcnUI

```bash
npx shadcn@latest init
npx shadcn@latest add button input textarea avatar badge dialog separator
```

### 3. Set up Clerk

1. Create an app at [clerk.com](https://clerk.com)
2. Copy `.env.local.example` → `.env.local` and fill in your Clerk keys

### 4. Set up Convex

```bash
npx convex dev
```

In **Convex Dashboard → Settings → Environment Variables** add:
```
CLERK_JWT_ISSUER_DOMAIN=https://your-clerk-issuer.clerk.accounts.dev
```

In **Clerk Dashboard → JWT Templates** create a new template named `convex`.

### 5. Run

```bash
# Terminal 1
npx convex dev

# Terminal 2
npm run dev
```
